# Game Development Company Management System

## Overview
A full-stack web-based management system designed for a game development company.
The system helps manage internal operations such as developers, projects, and workflow coordination.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Database: PostgreSQL

## Core Features
- Employee management for game developers
- Secure authentication with password hashing
- Project and task organization
- Database-driven backend logic

## Project Structure
- frontend/: User interface
- backend/: Server, API, and database logic
